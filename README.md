# CS556-Performance-Modelling-Coding-Assignment-
Repository contains Simulation Scripts for Part1, Part2 and Part3 as well as the Data Visualization Script.

## Running the 3 Simulation Scripts
The simulation scripts for the three algorithms has been written in C++.
Before executing the files, make sure to have g++ installed in the system.
1. Open the terminal
2. Ensure the .cpp files are residing in the current directory of the terminal
3. Inside the terminal, type "g++ filenames.cpp -o output" for compiling the program. Here, "filenames" stands for either "Part1_StandardFIFO_HoLBlocking", "Part2_VirtualOutputQueue_OptimalSearch" and "Part3_VOQ_with_iSLIP_Scheduling"
4. To execute the program, type "./output"
5. Simulation Logs, Total Service Time and Backlog Data will get printed on the screen.


## Running the Data Visualization Script
This script has been written using Python, inside Jupyter Notebook.
Before executing the script, ensure to have Jupyter Notebook installed in the system.
1. In the terminal, type "jupyter notebook"
2. Once the notebook is launched, upload the file "Data_Visualization_Script.ipynb"
3. Run the script in the notebook using the 'run' button.
4. Two graphs will be displayed on the screen.
